# wanted-pre-onboarding-frontend
원티드 프리온보딩 인턴십 사전과제

## 1. 프로젝트 설치 및 실행 방법
**✏️ clone**
```
git clone https://github.com/tiramin/wanted-pre-onboarding-frontend.git
```
**✏️ install**
```
npm i
```
**✏️ start**
```
npm start
```

## 2. 기술 스펙
`React`, `styled-component`

## 3. 데모 영상
준비중

## 4. 경로
  - **`/`** : 메인화면
  - **`/signup`** :  회원가입
  - **`/signin`** : 로그인
  - **`/todo`** : Todo
