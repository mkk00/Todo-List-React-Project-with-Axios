## 1. 프로젝트 설치 및 실행 방법
**✏️ clone**
```
git clone https://github.com/tiramin/Todo-List-React-Project-with-Axios.git
```
**✏️ install**
```
npm i
```
**✏️ start**
```
npm start
```

## 2. 기술 스펙
`React`, `styled-component`

## 3. [배포 링크](simpletodolistofmkk.netlify.app/)
